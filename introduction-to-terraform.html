1. What is Terraform ?

Terraform is a tool for building, updating and versioning infrastructure on cloud. Terraform can be used to interact with Amazon Cloud Services and Google Cloud Services. The infrastructure which Terraform can manage include low-level components such as compute instances, storage and networking, as well as high-level component as DNS entries, SaaS features ...

Infrastructure as code

Infrastructure is described using a high-level configuration syntax. And it is easy to shared.

Execution Plan

All steps will be planed before applying. Execution plan show what Terraform will do when you apply the news/changes things in your infrastructure.

2. Common helpful command lines

Terraform init

terraform init intinialize news or existing Terraform configuration.

Terraform plan

terraform plan generates and show Execution Plan.

Terraform apply

terraform plan builds or update infrastructure.

For more information, running terraform --help or refer to https://www.terraform.io/docs/commands/index.html.

Basicly, you can run 3 above commands when building your new infrastructure and it is enough.
